# HueCloset – Access & Usage Guide

## Overview
HueCloset is a web-based outfit recommendation system that helps users generate visually cohesive and weather-appropriate outfits using color analysis and temperature-based filtering.

The application is hosted online and does not require local installation.

---

## Accessing HueCloset

You can access the application directly in your browser:

👉 https://huecloset.netlify.app/

### Requirements
- A modern web browser (Chrome, Safari, Firefox, or Edge)
- Internet connection

No downloads or setup are required.

---

## How to Use the Application

### Step 1: Open the Website
Navigate to:
https://huecloset.netlify.app/

---

### Step 2: Upload Clothing Items
- Upload images of clothing items (e.g., shirts, pants, jackets)
- Supported formats: `.jpg`, `.png`, `.webp`

---

### Step 3: Input Temperature
- Enter the current or desired temperature
- This helps the system filter appropriate clothing items

---

### Step 4: Generate Outfits
- Click the **Generate** or equivalent button
- The system will:
  - Extract dominant colors from each clothing item
  - Filter items based on temperature suitability
  - Rank outfit combinations based on color compatibility

---

### Step 5: View Results
- The system displays recommended outfits
- Each outfit includes:
  - Selected clothing items
  - A score indicating quality
  - A brief explanation (color harmony + weather suitability)

---

## Example Workflow

1. Upload:
   - Blue shirt  
   - Black pants  
   - White sneakers  

2. Enter temperature:
   - 60°F  

3. Click generate  

4. Output:
   - Coordinated outfit suggestion with score and reasoning  

---

## Troubleshooting

- **Images not uploading:**  
  Ensure the file format is supported and the file is not corrupted  

- **No outfits generated:**  
  Try adding more clothing items or adjusting the temperature  

- **Slow performance:**  
  Refresh the page or try a different browser
